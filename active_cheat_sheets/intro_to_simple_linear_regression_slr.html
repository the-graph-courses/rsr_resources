Intro to Simple Linear Regression (SLR)

A best-fit line for two numeric variables
Big idea

Linear regression draws a single straight / curved / dashed line through a cloud of points to summarize the relationship between two numeric / categorical / binary variables. We use it to describe and quantify / ignore / hide a relationship, then predict / invent / forget new values of Y.

The equation
ŷ = b0 + b1 · x
ŷthe predicted value of Y at a given x
b0the intercept / slope / midpoint: predicted ŷ when x = 0 / 1 / mean
b1the slope / intercept / midpoint: change in ŷ per 1 / 10 / 100-unit increase in x
Anatomy of the line
Slope (b1)

The slope (b1) equals the steepness / color / curvature of the line.

It gives how much ŷ changes per 1-unit increase in x, namely b₁ / b₀ / x units.

It tilts the line up / down / sideways for positive values, and up / down / sideways for negative values.

Intercept (b0)

The intercept (b0) equals the predicted value of Y at x = 0 / 1 / mean.

It marks where the line crosses the Y-axis / X-axis / origin.

It is only meaningful as a prediction when x = 0 falls inside / outside / above the observed data range.

How the best-fit line is chosen
Residual & least squares

A residual is the vertical gap from a point to the line:

residual = actual / slope / zero − predicted / average / doubled

To pick the best-fit line, we want residuals to be small. Simply summing raw residuals does not work because positives and negatives cancel / double / multiply to zero, so we square / cube / invert each one first. The best-fit line minimizes the sum of squared / raw / absolute residuals, a rule called least squares / minimum slope / zero residuals.

For the highlighted point, residual = 5.5 − 3.8 = 1.7 / -1.7 / 9.3
Using the model
Reading R's summary(model)
# Fit, then inspect model <- lm(y ~ x, data = df) summary(model) Estimate (Intercept) 2.00 x 3.00

The intercept lives in the row labeled (Intercept) / (Slope) / (Constant), and the slope lives in the row named after the predictor / response / constant. Both values come from the Estimate / Std. Error / Pr(>|t|) column. So here b₀ = 2 / 3 / 5 and b₁ = 2 / 3 / 5.

Make a prediction

Once b₀ and b₁ have been estimated, predicting ŷ for a new x is just substitution into the equation.

Using the estimates from the R output, predict ŷ when x = 5:

ŷ = 2 / 3 / 5 + 2 / 3 / 5 · 2 / 3 / 5 = 13 / 17 / 22
Watch out for
Extrapolation

The model only describes the range / slope / midpoint of data you observed. Predictions outside that range, or Y values you have never seen, often produce nonsense.

Association ≠ causation

A non-zero slope shows association / proof / nothing, not necessarily a causal / imaginary / weak relationship.

Non-linearity

If the true pattern is curved / straight / flat, a line fits poorly / perfectly / instantly. Always plot the data first.
