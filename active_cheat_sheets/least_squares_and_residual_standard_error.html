Least Squares and Residual Standard Error

How regression chooses the line, summarizes leftover error, and reports typical miss size
Big idea

In simple linear regression, least squares fits a linear model / table / curve to data by choosing the slope / color / axis and intercept / median / unit that make the total squared / absolute / linear prediction error as small / large / random as possible.

The optimization target
SSE = Σ(yi − ŷi)2

A residual (also called e) is actual / average / slope y value minus predicted / doubled / zero y value, written ei = yi − ŷi.

We square / ignore / flip residuals so positive and negative misses do not cancel, then add them to get SSE / RSE / df.

How the line is chosen
What least squares does

In the scatterplot shown, the teal line is the fitted model. A better-fitting line makes the red residual segments shorter, so the Sum of squared residuals or Sum of Squared Errors (SSE) goes down / up / missing.

The least-squares line is the line whose SSE is the smallest / largest / average achievable across all possible straight lines through the data.

Each red segment is one residual, e. The red square's area is e2. The sum of all such squared areas is SSE / RSE / R².
Reading residuals

# top of summary(model)

-0.52 -0.48 -0.07 0.34 0.71

A healthy residual distribution looks roughly symmetric about zero:

Median ≈ 0 / 1 / RSE → residuals balanced around the line
|Q1| ≈ |Q3 / Q1 / RSE|
|Min| ≈ |Max / Min / Median|

Strong asymmetry hints that a straight line is the wrong model / axis / unit (the true relationship may not be linear).

Residual sign and size

A point above the fitted line has a positive / negative / zero residual. A point below the line has a negative / positive / squared residual.

For baby #4, actual weight is 3.06 kg and predicted weight is 2.36 kg:

e = 3.06 / 2.36 / 0.49 − 3.06 / 2.36 / 0.49 = 0.70 / -0.70 / 5.42
From residuals to residual standard error
Interpreting R's RSE output
Residual standard error: 0.549 on 6 degrees of freedom

R's output for the RSE above means the predicted weights are typically off by about 0.549 / 6 / 8 kg / weeks / slopes.

With 8 babies, the 6 degrees of freedom come from 8 − 2 / 6 / 8. Higher RSE usually means a worse / perfect / causal fit.

RSE formula & interpretation

Residual standard error is the typical / largest / smallest size of a residual.

RSE = √SSE / df

For simple linear regression, df = n − 2 / 1 / n. This is because fitting the line "spends" 2 / 1 / n numbers from the data: the slope and the intercept / median / maximum.

Vocabulary, notation, and synonyms
A note on naming

This topic introduces many new terms and symbols, and some have multiple names. Below is a summary of some key terms:

ei Residual for the ith observation. Also written ri. SSE Sum of Squared Errors / Effects / Estimates. Also called RSS (Residual Sum of Squares). RSE Residual Standard Error. Sometimes labeled as just s. The typical miss in Y-units.
ŷi Predicted (fitted) value for observation i. b0, b1 Estimated intercept and slope / residual / error. Also written β̂0, β̂1.
