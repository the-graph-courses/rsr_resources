Standard Error and Confidence Intervals for the Slope

Measuring slope uncertainty and building a plausible range for the true slope
Sample slope vs. population slope

In simple linear regression, the sample slope b₁ / β₁ / SE is our estimate of the unknown population slope β₁ / b₁ / RSE.

Because of sampling variability, different random samples would give different / identical / unrelated fitted slopes. The standard error, SE(b₁ / y / x), measures the typical spread / sign / average of those sample slopes.

In real studies, we usually observe one sample / population / cohort, not the whole population / sample / cohort. So SE(b₁) estimates how much b₁ would vary, using residual scatter, sample size, and x-spread.

A confidence interval / point / set combines the slope estimate, its SE, and a t* / SSE / residual multiplier to give a plausible range / point / value for the true slope.

Sampling variability. Blue: population scatter and fitted lines from other possible samples. Rose: one observed sample and its regression line.
Standard error of the slope
SE(b1)  =  s √Σ (xi − x̄)2
Here s is the residual standard / squared / mean error (RSE), and Σ(xi − x̄)2 (often called Sxx / SSE / SST) measures the spread / mean / count of x values.

A large SE means the slope estimate would vary / shrink / stabilize a lot from sample to sample; a small SE means the slope estimate would vary little / a lot / never from sample to sample.

SE(b₁) gets larger when residual scatter is high, sample size is small, or x values are tightly clustered.

More / Less / Equal residual scatter
→ larger SE
Larger / Smaller / Equal sample size 
→ smaller SE
More / Less / Equal x-spread (Sxx)
→ smaller SE
Building the 95% CI

A 95% CI for the slope is b₁ / β₁ / SSE ± t* / z / x̄ × SE(b₁) / R² / s.

For small regression samples, t* comes from a t-distribution with df = n − 2 / 1 / n. For large samples, t* ≈ 1.96.

In practice, R does all of this for you: summary / str / mean(model) reports b₁ and SE(b₁), and confint / predict / anova(model) gives the CI directly.

From SE to confidence interval
Suppose t* = 2, b₁ = 5, and SE(b₁) = 1.5.
95% CI = 5 ± 2 × 1.5 = 5 ± 3 / 2 / 1.5 = (2 / 3 / 1, 8 / 7 / 9).

Interpretation: our best estimate of β1 is 5 / 2 / 8, but allowing for sampling / measurement / prediction error, β1 could plausibly be anywhere from 2 to 8. Because (2, 8) lies entirely above / below / around zero, we can rule out β1 = 0.

Significance and the zero-slope test

If zero lies outside / inside / on the edge of the 95% CI, the slope is statistically significant.

If zero lies inside / above / outside the CI, we cannot rule out the possibility of no linear relationship.

An interval like (−0.45, −0.20) is significant / not significant / inconclusive because the whole interval is below zero. An interval like (−0.55, 0.10) is not significant / significant / borderline because it contains zero.

Significant: entirely below zero
Not significant: crosses zero

Vocabulary, notation, and distinctions
b1 Sample / Population / Random slope estimate (computed from data). β1 True population / sample / linear slope; the fixed value we are trying to estimate. s Residual standard / squared / mean error; typical vertical scatter around the line. Sxx Σ(xi − x̄)2, a measure of the spread / mean / median of the x values. n Sample size / mean / range; larger n usually makes SE(b1) smaller. SE(b1) Standard error of the slope / intercept / residual; typical sample-to-sample spread in b1. t* Critical / Mean / Median value for the chosen confidence level. CI Confidence interval for the slope: b1 ± t* / z / F × SE(b1).
