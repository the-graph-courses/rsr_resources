Interpreting the Slope: t-Values and p-Values

How regression tests whether the slope is real or just noise, using the t-statistic, the null hypothesis, and the p-value
The t-Statistic: A Signal-to-Noise Ratio

The t-statistic asks: how big is the slope compared to the uncertainty / mean / median around it? It acts as a signal / noise / power-to-noise / signal / bias ratio.

t = b1SE
Signal: estimated slope (b1); how far the fit is from zero / one / mean
Noise: standard error (SE); typical sample-to-sample variability / bias / shape
Ratio: how many SEs / units / percents the slope sits from zero

When the slope is far from zero relative to its uncertainty, the absolute value of t is large / small / negative, and this counts as strong / weak / moderate evidence of a true linear association. Worked example: if b1 = 0.45 and SE = 0.15, then t = 0.45 / 0.15 = 3 / 0.3 / 0.67, meaning the slope sits 3 / 6 / 0.15 standard errors from zero.

The null world

The null hypothesis H0 says the true slope is zero / large / unknown, meaning x has no / a strong / a negative linear effect on y in the population.

Even when the true slope is zero, random samples still produce slightly positive or negative slopes just from chance / causation / design. The bell-shaped spread of these null-world t-values is called the null / alternative / sample distribution.

A big |t| would be surprising / expected / typical in the null world, so it counts as evidence against / for / about H0.

Interpreting the p-valueis each statement correct or a misconception?
"p is the probability of seeing data this extreme if H0 were true."
Right. p is P(data | H0), not P(H0 | data).
"A very small p-value proves that the effect is large and important."
Misconception. Small p only says the slope is unlikely to be zero. Even tiny effects reach p < 0.001 in a big sample. Always look at the effect size too.
"p > 0.05 means there is no effect."
Misconception. The data are just inconclusive. A bigger sample might detect a real effect.
"p is the probability that the null hypothesis is true."
Misconception. p assumes H0 is true, then asks how surprising the data are. It says nothing about P(H0).
Significance star codes

CI and p-value: two ways of reporting the same test

The 95% CI and the p-value at α = 0.05 always give the same / opposite / partial significance verdict: the CI excludes / includes / contains zero exactly when p falls below 0.05. They report the same test, but they show different things to the reader.

Option A: estimate + CI (preferred)
"Birth weight increased by 0.39 kg per gestational week (95% CI: 0.10, 0.68)."
✅ Effect size (0.39 kg per week)
✅ Significance (zero outside the CI)
✅ Precision of the estimate (width of the interval)
Option B: estimate + p-value
"Birth weight increased by 0.39 kg per gestational week (p = 0.017)."
✅ Effect size (0.39 kg per week)
✅ Significance (p < 0.05)
❌ Does not show precision; reader cannot see how wide the plausible range is.
