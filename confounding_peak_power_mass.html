Confounding: body mass vs relative peak power

Peak power in athletes (06_peak_power_bone) — relative_peak_power ~ body_mass_kg + sex

Author

Research Statistics with R — Health Sciences

Question

This sample comes from a study on vertical jump-derived peak power and tibial bone strength (Denys & Yingling, MSSE 2022; Dryad). relative_peak_power is peak power expressed relative to body size (higher = more “explosive” power for your mass).

Does body mass predict relative peak power? In the pooled sample the answer can look like “yes,” but men and women differ in both mass and how power is expressed relative to mass. A simple Y ~ mass slope can partly reflect sex rather than a pure mass effect.

Data

datasets_staging/cleaned/06_peak_power_bone.csv. Variables: relative_peak_power, body_mass_kg, sex (female / male).

d <- read_csv("datasets_staging/cleaned/06_peak_power_bone.csv", show_col_types = FALSE) |>
  mutate(sex = factor(sex, levels = c("female", "male"))) |>
  filter(!is.na(relative_peak_power), !is.na(body_mass_kg), !is.na(sex))

Regression: bivariate vs adjusted

mod_bi  <- lm(relative_peak_power ~ body_mass_kg, data = d)
mod_adj <- lm(relative_peak_power ~ body_mass_kg + sex, data = d)

bind_rows(
  tidy(mod_bi)  |> mutate(model = "Bivariate:  rel_peak_power ~"),
  tidy(mod_adj) |> mutate(model = "Adjusted:     rel_peak_power ~")
) |>
  filter(term %in% c("(Intercept)", "body_mass_kg", "sexmale")) |>
  transmute(model, term, estimate, std.error, statistic, p.value) |>
  knitr::kable(digits = 3, caption = "Focus on the slope for `body_mass_kg`")

The bivariate slope on mass is clearly positive and significant. After + sex, the slope is much smaller and only marginally significant at conventional levels — the mass effect in the pooled model was substantially diluted once sex is controlled. (Physiologically, relative power and mass need not follow a single pooled straight line across sexes; this lesson stresses confounding / composition, not a final causal claim.)

Compare to the multiple-regression line

The adjusted model assumes one slope for body_mass_kg and parallel lines across sex (different intercepts). You can overlay those fitted lines to connect the plots to summary(mod_adj):

mass_seq <- seq(min(d$body_mass_kg), max(d$body_mass_kg), length.out = 80)
d_pred <- expand.grid(
  body_mass_kg = mass_seq,
  sex = levels(d$sex),
  stringsAsFactors = FALSE
)
d_pred$sex <- factor(d_pred$sex, levels = levels(d$sex))
d_pred$fit <- predict(mod_adj, newdata = d_pred)

ggplot(d, aes(body_mass_kg, relative_peak_power, colour = sex)) +
  geom_point(alpha = 0.35, size = 2) +
  geom_line(data = d_pred, aes(y = fit, colour = sex), linewidth = 1) +
  scale_colour_manual(values = pal, name = "Sex") +
  labs(
    title = "Adjusted model (parallel slopes)",
    subtitle = "Lines = predictions from lm(relative_peak_power ~ body_mass_kg + sex)",
    x = xlab, y = ylab
  )

Take-home

  1. relative_peak_power ~ body_mass_kg can show a strong positive pooled slope.
  2. relative_peak_power ~ body_mass_kg + sex pulls the mass coefficient toward zero — much of the pooled association was entangled with male–female differences.
  3. Body mass is the “size” variable here (there is no height column in this file). The logic is the same as height vs a performance measure: sex tracks both size and outcome, so it is an obvious candidate confounder in teaching examples.
