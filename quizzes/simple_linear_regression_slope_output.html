The least squares line is the line that:

Least squares chooses the line that minimises the sum of squared residuals. Residuals are the vertical gaps between observed and predicted y values.

What does the standard error of the slope estimate?

The SE of the slope estimates how much the slope would vary if we repeatedly sampled from the same population and refit the model.

The 95% CI for the beers slope is 0.013 to 0.023. What does this imply?

A 95% CI that excludes zero gives a significant two-sided slope test at the 5% level. Because this interval is fully above zero, the association is positive.

What model is this R code fitting?

model <- lm(weight_kg ~ weeks, data = preemie_data)
The formula y ~ x means model y as a function of x. Here, weight_kg is the outcome and weeks is the predictor.

What are these two R commands mainly asking for?

summary(model)
confint(model)
summary(model) gives the coefficient table with estimates, standard errors, t values, and p-values. confint(model) gives confidence intervals for the coefficients.

A linear regression on 100 data points gives slope = 0.15 and SE = 0.20. Using t = slope / SE, is the slope likely significant at the 5% level?

With 100 data points, a rough rule is that |t| needs to be around 2 or larger for p < 0.05. Here t = 0.15 / 0.20 = 0.75, so this would not be significant.

A study with n = 50,000 reports a slope of 0.002 with p < 0.001. What is the best interpretation?

A small p-value is evidence against a zero slope, but it does not by itself say that the effect is large. Large samples can make tiny effects statistically significant.

A study finds p = 0.03 for the slope relating weekly exercise to resting heart rate.

Part 1: What is the null hypothesis being tested here?

The null hypothesis for a regression slope test is that the true slope equals zero — i.e. no linear association between predictor and outcome in the population.

Part 2: Which interpretation of p = 0.03 is correct?

A p-value is computed assuming the null is true. So p = 0.03 means results this extreme would occur about 3% of the time if the slope were truly zero — not the probability that the null itself is true.

Look at this 95% confidence interval for a slope. What conclusion matches it?

The CI runs from -0.10 to 0.30, so it crosses zero. A 95% CI that includes zero is not significant at the 5% level.
